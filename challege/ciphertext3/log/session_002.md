2. 二重Vigenere、Gromark、オートキーなど各種暗号方式の検証

4. 最終復号

## 試行と結果

### 1. Vigenere鍵長361
- フリードマン: 周期361のストリームIC = 0.042（ランダムレベル）
- 各ストリーム18文字と少なく頻度分析が困難
- 復号結果のIC = 0.050（改善不十分）
- 鍵グリッドに構造なし → 棄却

### 2. 二重Vigenere（key1[c] + key2[r]）
- ガウス: 19x19の加法構造を仮説
- mutual ICで行シフトを推定するも信号が弱い
- 復号IC = 0.040 → 棄却

### 3. 進行型Vigenere（delta 0-25）
- ダ・ヴィンチ: 各ブロックで鍵全体がdeltaずつシフトする仮説
- 全delta値でIC = 0.039 → 棄却

### 4. Hill climbing（Vigenere-19, Gromark各種, Beaufort）
- チューリング: bigram scoringベースのsimulated annealing
- Vigenere-19: IC = 0.039
- Gromark（k[i-1]+k[i-19]）: IC = 0.039
- Gromark（k[i-19]+k[i-18]）: IC = 0.039
- 全て棄却

### 5. フィードバック/差分暗号
- ct[i]-ct[i-k] for k=1..39: 全てIC = 0.039
- ct[i]+ct[i-k]: 全てIC = 0.039
- 二重差分: IC = 0.039
- 全て棄却

### 6. Cross-IC分析
- 観測値が理論的cross-ICより一貫して高い
- 2段ローター仮説も棄却

### 7. 平文オートキー再検討
- シャノン: 畳み込みICを理論計算 → 0.0397（ランダムとほぼ同じ）
- 観測ストリームIC 0.044との差が大きい
- Hill climbing復号もIC = 0.039 → 棄却

### 8. 列転置 + Vigenere（積暗号）→ 成功！
- ダ・ヴィンチ: 転置解除後にストリームICが回復する仮説
- 幅2-8で全順列探索を実施
- **幅7, 列順[4,2,3,0,6,5,1]でストリームIC = 0.068に回復！**
- 頻度分析でVigenere鍵 = LIBERTYEAGLEJUSTICE を即座に特定
- 平文 = アメリカ独立宣言（全文）

## 解読結果
- **暗号方式**: Vigenere（鍵LIBERTYEAGLEJUSTICE, 周期19）+ 列転置（幅7）
- **平文**: アメリカ独立宣言
- **平文IC**: 0.06815

## 教訓
- ストリームICが理論値（0.067）に達しない場合、単一の暗号方式ではなく積暗号を疑うべき
- 転置暗号はIC自体を変えないが、周期的暗号のストリーム構造を破壊する
- 小さい転置幅（7以下）なら全順列探索が実用的（7! = 5040通り）
